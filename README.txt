Para la ejecución de la tarea se debe abrir un terminal en la carpeta actual y escribir el comando

	make

Para la compilación del código perteneciente al servidor.

Luego se debe iniciar el servidor con el comando:

	make run

Para la compilación y ejecución del cliente, basta abrir otro terminal en esta carpeta y escribir en consola:

	make cliente

Y procedera a dar inicio al cliente del FTP.

Para las máquinas a usar se debe copiar la carpeta "Maquina" (disponible en src) en la maquina de destino
y luego dentro de estas carpetas ejecutar los comandos siguientes:

	make

	make run

 java 1.8+
